多容器部署：
	#创建一个网络
	1. docker network create -d bridge my-bridge
	# 查看一下网络
	# docker network ls
	# 创建容器 连接到my-bridge 网络上
	2. docker run -d --name test1 --network my-bridge redis
	# 将已经启动的容器，连接到my-bridge上面
	# docker network connect my-bridge test2
	# 此时test1 test2 相互都能ping通
